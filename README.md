## ✨ Features

### 📖 Academic Content
| Feature | Description |
|---------|-------------|
| 📄 Notes | Unit-wise notes for all FE subjects |
| 📋 PYQs | Past Year Question Papers 2021–2024 |
| 📝 Unit Quizzes | MCQ with instant results & explanations |
| 📖 Syllabus | Admin can add/edit subjects & units |

### 🤖 AI Tools
| Feature | Description |
|---------|-------------|
| 🤖 Study Bot | Generate assignments, notes & quizzes |
| 🎤 Voice Search | Speech to text input |
| ✨ Gemini AI | One-tap Google Gemini integration |
| 📅 Study Plan | AI generated 7-day study schedule |

### 🛠 Study Tools
| Feature | Description |
|---------|-------------|
| 📋 Planner | Tasks with priority & deadlines |
| 📅 Calendar | Exam & event tracking |
| 💬 Forum | Student & teacher Q&A |
| 🔖 Bookmarks | Save notes & PYQs |
| 🏆 Leaderboard | Top contributors ranking |
| 🔍 Search | Advanced search with filters & voice |

### 📢 Announcements
| Feature | Description |
|---------|-------------|
| 🔗 Links | Clickable external resource links |
| 📌 Pin | Pin important notices |
| 🔴 Unread | Badge & notification bell |
| 📊 Analytics | View count tracking |

## ⚙️ Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/Rajeshwar20/sppu-study-hub3.git

# 2. Go into the folder
cd sppu-study-hub3

# 3. Install dependencies
npm install

# 4. Start the app
npm start
```

✅ Open **http://localhost:3000** in your browser

## 🚀 Deploy Your Own Copy

### Option 1 — Vercel (Recommended)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Rajeshwar20/sppu-study-hub3)

1. Click button above
2. Connect GitHub account
3. Click **Deploy**
4. Live in 2 minutes! 🎉

### Option 2 — GitHub Pages

```bash
npm install gh-pages --save-dev
npm run deploy
```
